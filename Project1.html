Project 1

Vending Machine Code
This program simulates a vending machine where users can make purchases of various snacks and drinks.
- The program provides a menu of available items along with their prices and stock levels.

- Users can enter the amount of money they have and make selections.
- The program checks if the user has sufficient funds and if the selected item is in stock, then deducts the cost from the user's balance.

- It updates the stock levels and transaction history.
- The program also suggests additional purchases based on the user's selection.

• The Vending Machine has a variety of snacks

• The Vending Machine has a system of choosing the snack and also suggesting an item with it while also checking the amount

• After selection the user is suggested with an item while checking the balance
